# Skills 사용 방법

스킬은 봇에서 **기능별 사용법**을 마크다운(.md)으로 저장해 두는 공간입니다.

## 명령어 (모두 `/` 로 시작)

| 명령 | 설명 |
|------|------|
| `/skills` | 이 사용 방법 문서 보기 |
| `/skills <이름>` | 해당 스킬 문서 내용 보기 |
| `/addskills` | 새 스킬 추가 (이후 채팅으로 이름·설명 입력) |
| `/skill list` | 등록된 스킬 목록 |
| `/skill remove <이름>` | 스킬 삭제 |

## 예시

- `/skills` → 이 문서가 출력됩니다.
- `/skills 날씨` → `날씨.md` 내용이 출력됩니다.
- `/addskills` 입력 후, 봇이 "스킬 이름을 보내주세요"라고 하면 이름을 보내고, 이어서 "기능 설명을 보내주세요"에 설명을 채팅으로 보내면 `이름.md` 파일로 저장됩니다.
